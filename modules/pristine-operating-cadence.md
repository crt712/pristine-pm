# Pristine Operating Cadence (Canonical)

Use this cadence to protect signal quality while building.

## Weekly (30-60 min)
- Check for drift in formula, copacker readiness, margin assumptions, and message quality.
- If a risk moved from possible to probable, escalate immediately in `active/pristine/STATUS.md`.

## Monthly
- Review product integrity (stability/testing signal), margin integrity, and message integrity.
- Correct weakness before adding scope.

## Quarterly
- Evaluate whether acceleration is earned:
- performance durability
- margin stability
- repeat-purchase signal
- operational consistency

## Rule
- Expansion follows proof.
- If durability weakens, expansion pauses.
