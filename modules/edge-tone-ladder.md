# Tone / Edge Ladder (Canonical)

Use this ladder to choose how you show up in writing and real-time conversations. The goal is not harshness; it's **clarity + intention** with the minimum force required to protect outcomes.

## Levels

### Level 1 — Natural (Structured & collaborative)
- Thoughtful, clear, low pressure, forward-moving
- Default mode when things are on track

Example language:
- "We're aiming to have this wrapped by [day]. If you foresee any challenges, let me know early so we can adjust."

### Level 2 — Intentional (More explicit expectations)
- Earlier directness, clearer next steps, defined timelines
- Use when ambiguity is creeping in

Example language:
- "To keep this moving, I need an update by [time/day] and a clear next step."

### Level 3 — Sharpened (Clear + outcome-driven, decision point)
- Minimal ambiguity; strong accountability; calm delivery
- Use when patterns emerge (slippage, non-responsiveness, imbalance)

Example language:
- "At this point, I need a clear answer on how we're moving forward."

### Level 4 — Controlled authority (Firm control, used sparingly)
- No cushioning; decisive; still calm (no anger)
- Use when clarity has already been provided and impact is meaningful

Example language:
- "We're past the point of ambiguity. This needs to be resolved today. If that's not realistic, tell me now."

## Triggers to level up
- Lack of responsiveness
- Repeated slippage
- Low effort or poor quality
- Imbalance in contribution
- Feeling taken advantage of

Internal trigger:
- Escalate when: **the behavior signals a decision that negatively impacts the outcome**.

## Guardrails (what this is not)
- Not aggressive, confrontational, transactional, or emotional escalation
- Tone stays: **calm, precise, controlled**
